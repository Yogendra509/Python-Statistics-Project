#Salary_Data_Analysis 

This project focuses on analyzing salary data to identify trends based on factors such as job role, etc. The dataset was cleaned, processed, and analyzed using Python libraries.

Tools used :
Python,
Pandas,
Matplotlib,
Jupyter Notebook (via Anaconda)

Key Features :
Data cleaning and preprocessing,
Average, median, and standard deviation calculation,
Experience vs Salary analysis,
Job role-wise salary comparison,
Data visualization using charts and graphs,
Insights and conclusions from salary trends.
